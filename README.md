# Formula-Forum

## Proje Hakkında
<ul>
 <li>Teknoloji alanında yaşadığımız sorunlara formül bulmak için kullanabileceğimiz bir forum sitesi </li>
 <li>Projeyi geliştirirken .Net Mvc ile solid prensiplerine uygun bir şekilde N tier arc. olarak adlandırılan n katmanlı mimari yapısını bozmadan ilerlemeye çalıştım.</li>
 <li>Projenin frontend tarafında Html5, Css, Bootstrap, Javascript, Jquery bileşenlerine yer vererek tasarım tarafını gerçekleştirdim.</li>
 <li> Projemde veri tabanı olarak MSSQL kullandım, sunucuya taşırken windows hosting desteği olan bir sunucu üzerinde çalıştım.</li>
 <li>Veri tabanını oluştururken Entity Framework Code First yaklaşımını kullandım.</li>
 <li>Projemi Business, Entity, Data Access ve Presentation olmak üzere 4 temel katmanda geliştirdim.</li>
</ul>
